# 🛡️ C++ Directory Scanner (Security Tool)

Simple C++ program that scans a directory and detects potentially suspicious files based on their extensions (e.g. `.exe`, `.bat`).

---

## 🚀 Features

* 📁 Scan a directory from user input
* ⚠️ Detect suspicious files (`.exe`, `.bat`)
* 🖥️ Display results in the console
* 🧠 Built using `std::filesystem`

---

## 🛠️ Technologies

* C++
* Visual Studio
* `std::filesystem`

---

## ▶️ How to use

1. Run the program
2. Enter the path of the directory to scan
3. The program analyzes the folder and displays suspicious files

---

## 📌 Example

```txt
=== Mini Directory Scanner ===

Veuillez entrer le chemin du dossier à scanner :
C:\Users\Quentin\Desktop\test

⚠️ Fichier suspect détecté : virus.exe
⚠️ Fichier suspect détecté : script.bat
```
